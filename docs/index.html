Features

Everything you need, nothing you don't

Designed to disappear into your workflow — all orchestration happens silently.

🔌

Drop-in Plugin

Works in any Claude Code project without extra .mcp.json setup. Plugin manifest loads all MCP servers automatically.

🎯

Silent Auto-Routing

Opus classifies every task and picks the cheapest capable model. Never asks which model to use — it just decides.

⚡

Parallel Execution

Independent subtasks dispatched in a single message. Multi-file edits, searches, and renders run concurrently across providers.

🛡️

Verification Gate

Codex reviews every non-trivial diff before completion. If it flags issues, Sonnet/Haiku fix them — Opus never skips this step.

💰

Cost Discipline

Haiku before Sonnet. Sonnet before Ollama. Ollama before frontier NIM. Cheapest capable model wins, every time.

🔒

Security Auditing

NVIDIA Security pipeline with LLaMA Guard, Nemotron, Granite Guardian for SAST, PII detection, and threat modeling.

Installation

Get running in 2 minutes

Clone, install, and start using all 44+ models in any Claude Code project.

Quick Start
# 1. Clone the repository
git clone https://github.com/ranjankumarpatel/claude-code-multi-model
cd claude-code-multi-model

# 2. Install the plugin
npm install

# 3. Add required API keys to your environment
export NVIDIA_API_KEY="your-nvidia-key"   # For NVIDIA NIM/Security
export GEMINI_API_KEY="your-gemini-key"   # For Google Gemini (or OAuth)
export GH_TOKEN="your-github-token"       # For GitHub Copilot CLI

# 4. Open Claude Code — multi-model routing is active automatically
claude

Anthropic models work out-of-the-box with your existing Claude Code auth. Ollama cloud, Gemini, and opencode have free tiers — only NVIDIA NIM and Copilot CLI require paid access. See MCP-SETUP.md for detailed provider configuration.
